üß∞ 4. Using RestTemplate (Legacy, but Interview Favorite)

Example:

RestTemplate restTemplate = new RestTemplate();

String url = "http://localhost:8082/products/1";

Product product = restTemplate.getForObject(url, Product.class);

Internals of getForObject():

  • Uses HttpURLConnection under the hood via SimpleClientHttpRequestFactory
  • Performs a 3-way TCP handshake
  • Sends the request
  • Waits for response
  • Uses Keep-Alive cache to reuse TCP connections

üîç 6. What is Keep-Alive?

  • Default in HTTP/1.1
  • Reuses TCP connection for multiple HTTP requests
  • Controlled by headers: 
    Connection: keep-alive
Keep-Alive: timeout=5, max=50
  • Saves time from repeated handshakes

‚öôÔ∏è 7. Methods in RestTemplate

üß™ exchange() Example:

Product product = new Product("Ice-cream", 100);
HttpEntity<Product> requestEntity = new HttpEntity<>(product, headers);

ResponseEntity<Product> response = restTemplate.exchange(
    "http://localhost:8082/products",
    HttpMethod.POST,
    requestEntity,
    Product.class
);

üõ†Ô∏è 8. Advanced: Manual execute() with Custom Callbacks

RequestCallback requestCallback = request -> {
    request.getHeaders().setContentType(MediaType.APPLICATION_JSON);
    byte[] body = new ObjectMapper().writeValueAsBytes(new Product("Ice-cream", 100));
    StreamUtils.copy(body, request.getBody());
};

ResponseExtractor<String> responseExtractor = response ->
    StreamUtils.copyToString(response.getBody(), StandardCharsets.UTF_8);

String response = restTemplate.execute(
    "http://localhost:8082/products",
    HttpMethod.POST,
    requestCallback,
    responseExtractor
);
